## 3. How the button should work (Stripe logic)

Each article button contains a **Stripe Price ID**:

```html

  Acc√©der
</button>
```

Your JS (in `my-worker`) should:

1. Catch the click
2. Send the `price_id` to your Node server
3. Redirect to Stripe Checkout

### Example `stripe.js` (simplified)

```js
document.querySelectorAll(".premium-access").forEach(button => {
  button.addEventListener("click", () => {
    fetch("/create-checkout-session", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        priceId: button.dataset.priceId
      })
    })
    .then(res => res.json())
    .then(data => {
      window.location.href = data.url;
    });
  });
});
```
