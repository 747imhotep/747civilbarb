📌 Comment utiliser ce tableau de bord ?

  • Sélectionner un article → Choisissez le texte sur lequel vous travaillez.
  • Voir le document → Ouvre le PDF/TXT à lire ou modifier.
  • Mettre à jour → Indiquez combien de mots vous avez écrits aujourd'hui. La barre de progression se mettra à jour.
  • Envoyer ma révision → Envoyez votre version révisée du document par email à l'équipe éditoriale.
  • Documents récents → Liste des fichiers que vous avez déjà envoyés.

📌 How to use this dashboard?

  • Select an article → Choose the text you are working on.
  • View the document → Opens the PDF/TXT to read or modify.
  • Update → Indicate how many words you have written today. The progress bar will update.
  • Send my revision → Send your revised version of the document by email to the editorial team.
  • Recent Documents → List of files you have already sent.
